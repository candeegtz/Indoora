## Indoora - Aplicación para sisteema indoor

Las tecnologías del procesamiento de texto como **TTS (Text-to-Speech)** y **ASR (Automatic-Speech-Recognition)** presentan una ayuda que a simple vista es imprescindible; no obstante, para muchas personas su presencia es considerada una prioridad. La facilidad y comodidad que ofrecen estas herramientas se presentan en distintos ámbitos, y estas han ido evolucionando hasta el punto de responder simulando una conversación con un humano.

Este **Trabajo de Fin de Grado** incorpora el procesamiento de texto como automatización para la configuración de un sistema de posicionamiento en interiores. Dicha solución fue diseñada para conocer la localización y las actividades de personas con cierto grado de dependencia sin invadir la privacidad. 

Para que el sistema sea accesible a todo tipo de público, se propone desarrollar una aplicación móvil que facilite la configuración del sistema y permita la creación de rutinas y recordatorios, entre otras funcionalidades adicionales.
